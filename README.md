# Класс GeoCoordinate

## Описание
Класс `GeoCoordinate` представляет географические координаты, состоящие из значений широты и долготы.

## Атрибуты
- `geo_latitude` (float): Значение широты географической координаты.
- `geo_longitude` (float): Значение долготы географической координаты.

## Методы
### `__init__(self, geo_latitude: float, geo_longitude: float)`
Инициализирует новый экземпляр класса `GeoCoordinate`.

**Параметры:**
- `geo_latitude` (float): Значение широты географической координаты.
- `geo_longitude` (float): Значение долготы географической координаты.

**Исключения:**
- `ValueError`: Если значение широты или долготы отрицательное или превышает допустимый диапазон (0-180 градусов).

**Пример:**
```python
from geo_coordinate import GeoCoordinate

# Создание нового экземпляра GeoCoordinate
coordinate = GeoCoordinate(40.730610, -73.935242)
print(coordinate.geo_latitude)  # Вывод: 40.73061
print(coordinate.geo_longitude)  # Вывод: -73.935242
```

## Зависимости
Этот класс зависит от функции `check_data_type` из модуля `check_data_type`.

## Использование
Класс `GeoCoordinate` может использоваться для представления и работы с географическими координатами в вашем приложении. Он предоставляет способ создания и управления этими координатами, обеспечивая, что значения находятся в допустимом диапазоне.

# Класс Attraction

## Описание
Класс `Attraction` представляет собой географическую достопримечательность, такую как памятник или объект интереса. Он содержит информацию о названии достопримечательности, географических координатах и дате постройки.

## Атрибуты
- `naming` (str): Название достопримечательности.
- `coordinate` (`GeoCoordinate`): Географические координаты достопримечательности.
- `date_of_building` (int): Год постройки достопримечательности.

## Методы
### `__init__(self, naming: str, coordinate: GeoCoordinate, date_of_building: int)`
Инициализирует новый экземпляр класса `Attraction`.

**Параметры:**
- `naming` (str): Название достопримечательности.
- `coordinate` (`GeoCoordinate`): Географические координаты достопримечательности.
- `date_of_building` (int): Год постройки достопримечательности.

**Исключения:**
- `ValueError`: Если дата постройки отрицательная или равна нулю.

**Пример:**
```python
from geo_coordinate import GeoCoordinate
from attraction import Attraction

# Создание нового экземпляра GeoCoordinate
coordinate = GeoCoordinate(40.730610, -73.935242)

# Создание нового экземпляра Attraction
attraction = Attraction("Статуя Свободы", coordinate, 1886)
print(attraction.naming)  # Вывод: Статуя Свободы
print(attraction.coordinate.geo_latitude)  # Вывод: 40.73061
print(attraction.coordinate.geo_longitude)  # Вывод: -73.935242
print(attraction.date_of_building)  # Вывод: 1886
```

## Зависимости
Этот класс зависит от класса `GeoCoordinate` из модуля `geo_coordinate` и функции `check_data_type` из модуля `check_data_type`.

## Использование
Класс `Attraction` может использоваться для представления и управления информацией о географических достопримечательностях в вашем приложении. Он предоставляет способ создания и работы с этими достопримечательностями, обеспечивая корректность даты постройки.

# Класс DateBase

## Описание
Класс `DateBase` представляет собой базу данных для хранения информации о достопримечательностях и обратной связи по ним. Он содержит методы для получения информации о достопримечательности и публикации обратной связи.

## Атрибуты
- `length_of_text1` (int): Длина текста для информации о достопримечательности, если год постройки меньше или равен 1000.
- `length_of_text2` (int): Длина текста для информации о достопримечательности, если год постройки больше 1000.
- `attraction_to_information` (dict): Словарь для хранения информации о достопримечательностях.
- `attraction_to_feedback` (dict): Словарь для хранения обратной связи по достопримечательностям.

## Методы
### `get_information(attraction: Attraction) -> str`
Возвращает информацию о достопримечательности.

**Параметры:**
- `attraction` (`Attraction`): Объект достопримечательности.

**Возвращает:**
- `str`: Информация о достопримечательности.

### `feedback_publication(attraction: Attraction, text: str)`
Публикует обратную связь по достопримечательности.

**Параметры:**
- `attraction` (`Attraction`): Объект достопримечательности.
- `text` (str): Текст обратной связи.

## Использование
Класс `DateBase` может использоваться для хранения информации о достопримечательностях и обратной связи по ним. Метод `get_information` позволяет получить информацию о достопримечательности, а метод `feedback_publication` - опубликовать обратную связь.

Пример использования:
```python
from attraction import Attraction
from datebase import DateBase

# Создание объекта достопримечательности
attraction = Attraction("Эйфелева башня", coordinates, 1889)

# Получение информации о достопримечательности
information = DateBase.get_information(attraction)

# Публикация обратной связи
DateBase.feedback_publication(attraction, "Прекрасное место для посещения!")
```

# Класс Path

## Описание
Класс `Path` представляет собой путь между двумя географическими координатами. Он содержит информацию о начальной и конечной координатах, а также рассчитывает время движения различными способами передвижения.

## Атрибуты
- `coordinate_start` (`GeoCoordinate`): Начальная географическая координата.
- `coordinate_finish` (`GeoCoordinate`): Конечная географическая координата.
- `distance` (float): Расстояние между начальной и конечной координатами.
- `time_movement_on_foot` (float): Время движения пешком.
- `time_movement_on_bike` (float): Время движения на велосипеде.
- `time_movement_on_bus` (float): Время движения на автобусе.
- `time_movement_on_motorbike` (float): Время движения на мотоцикле.
- `time_movement_on_car` (float): Время движения на автомобиле.

## Методы
### `__init__(self, coordinate_start: GeoCoordinate, coordinate_finish: GeoCoordinate)`
Инициализирует новый экземпляр класса `Path`.

**Параметры:**
- `coordinate_start` (`GeoCoordinate`): Начальная географическая координата.
- `coordinate_finish` (`GeoCoordinate`): Конечная географическая координата.

## Использование
Класс `Path` может использоваться для представления и расчета времени движения между двумя географическими координатами различными способами передвижения. Он предоставляет информацию о расстоянии и времени, необходимом для преодоления пути.

Пример использования:
```python
from geo_coordinate import GeoCoordinate
from path import Path

# Создание начальной и конечной координат
start_coordinate = GeoCoordinate(40.730610, -73.935242)
finish_coordinate = GeoCoordinate(48.856613, 2.352222)

# Создание объекта пути
path = Path(start_coordinate, finish_coordinate)

# Вывод информации о времени движения
print(f"Время движения пешком: {path.time_movement_on_foot:.2f} часов")
print(f"Время движения на велосипеде: {path.time_movement_on_bike:.2f} часов")
print(f"Время движения на автобусе: {path.time_movement_on_bus:.2f} часов")
print(f"Время движения на мотоцикле: {path.time_movement_on_motorbike:.2f} часов")
print(f"Время движения на автомобиле: {path.time_movement_on_car:.2f} часов")

**Исключения:**
- `ValueError`: Если указанной координаты нет на карте.

## Использование
Класс `GeoMap` позволяет управлять картой достопримечательностей, добавляя и удаляя информацию о достопримечательностях.

# Класс Photo

## Описание
Класс `Photo` представляет собой фотографию достопримечательности. Он содержит информацию о достопримечательности, высоте, ширине и уровне красоты фотографии.

## Атрибуты
- `attraction` (`Attraction`): Достопримечательность, к которой относится фотография.
- `height` (int): Высота фотографии.
- `width` (int): Ширина фотографии.
- `beautiful_level` (int): Уровень красоты фотографии.

## Методы
### `__init__(self, attraction: Attraction, height: int, width: int, beautiful_level: int)`
Инициализирует новый экземпляр класса `Photo`.

**Параметры:**
- `attraction` (`Attraction`): Достопримечательность, к которой относится фотография.
- `height` (int): Высота фотографии.
- `width` (int): Ширина фотографии.
- `beautiful_level` (int): Уровень красоты фотографии.

**Исключения:**
- `ValueError`: Если высота или ширина отрицательные или равны нулю.

### `browsing_photo(self) -> Attraction`
Возвращает объект достопримечательности, к которой относится фотография.

**Возвращает:**
- `Attraction`: Объект достопримечательности.

## Использование
Класс `Photo` позволяет представлять и работать с фотографиями достопримечательностей. Он содержит информацию о фотографии и метод для просмотра достопримечательности, к которой относится фотография.

Пример использования:
```python

from photo import Photo

# Создание объекта достопримечательности
attraction = Attraction("Эйфелева башня", coordinates, 1889)

# Создание объекта фотографии

# Получение информации о достопримечательности на фотографии

# Класс ElectronicGuide

## Описание
Класс `ElectronicGuide` представляет собой электронный гид, который обеспечивает информацию о достопримечательностях, фотографиях, путях и позволяет взаимодействовать с ними.

## Методы
### `__init__(self, list_of_attraction: list, current_coordinate: GeoCoordinate)`
Инициализирует новый экземпляр класса `ElectronicGuide`.

**Параметры:**
- `list_of_attraction` (list): Список достопримечательностей.
- `current_coordinate` (`GeoCoordinate`): Текущие географические координаты.

### `browsing_photo(self, naming: str) -> Photo`
Просмотр фотографии по названию достопримечательности.

**Параметры:**
- `naming` (str): Название достопримечательности.

**Возвращает:**
- `Photo`: Объект фотографии.

### `get_path(self, naming_start: str, naming_finish: str) -> Path`
Получение пути между двумя достопримечательностями.

**Параметры:**
- `naming_start` (str): Название начальной достопримечательности.
- `naming_finish` (str): Название конечной достопримечательности.

**Возвращает:**
- `Path`: Объект пути.

### `movement(self, next_naming: str)`
Перемещение к следующей достопримечательности.

**Параметры:**
- `next_naming` (str): Название следующей достопримечательности.

### `get_information(self, naming: str) -> str`
Получение информации о достопримечательности.

**Параметры:**
- `naming` (str): Название достопримечательности.

**Возвращает:**
- `str`: Информация о достопримечательности.

### `feedback_publication(self, text: str)`
Публикация обратной связи по текущей достопримечательности.

**Параметры:**
- `text` (str): Текст обратной связи.

## Использование
Класс `ElectronicGuide` предоставляет функциональность электронного гида, позволяя получать информацию о достопримечательностях, просматривать фотографии, перемещаться между ними, получать пути и публиковать обратную связь.

Пример использования:
```python
from attraction import Attraction
from geo_coordinate import GeoCoordinate
from electronic_guide import ElectronicGuide

# Создание объекта электронного гида
list_of_attraction = [attraction1, attraction2, attraction3]
current_coordinate = GeoCoordinate(0, 0)
electronic_guide = ElectronicGuide(list_of_attraction, current_coordinate)

# Просмотр фотографии
photo = electronic_guide.browsing_photo("Название достопримечательности")

# Получение пути
path = electronic_guide.get_path("Начальная достопримечательность", "Конечная достопримечательность")

# Перемещение к следующей достопримечательности
electronic_guide.movement("Следующая достопримечательность")

# Получение информации о достопримечательности
information = electronic_guide.get_information("Название достопримечательности")

# Публикация обратной связи
electronic_guide.feedback_publication("Текст обратной связи")
